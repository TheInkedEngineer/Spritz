generateFiscalCode(from:)

Generates the Codice Fiscale from the given information.

Declaration

static func generateFiscalCode(from info: SpritzInformationProvider) throws -> CodiceFiscale

Return Value

A 16 letter long string representing the fiscal code based on the provided data.

Parameters

info

An object adhering to SpritzInformationProvider protocol providing the necessary data.

Discussion

Note

Spritz.Error.invalidData if the data is not valid. That can occur if an unsupported character was passed, or the place of birth was invalid.
